### 🚀 Things I've built

**[VaultRelay](https://github.com/arihanthsharma15/vaultrelay)** ([Docs](https://github.com/arihanthsharma15/vaultrelay/tree/main/docs)) — Zero-trust NL-to-SQL middleware. Guardian (cloud gateway) + Sentry (local agent) communicating over an outbound-only HMAC-signed tunnel — no inbound firewall rule ever needed on the client side. Schema-aware SQL generation, dual-layer SELECT-only validation, PII redaction, audit logging. 77 passing tests across both services.

**[Clinq AI](https://github.com/arihanthsharma15/cliniq-ai)** ([Docs](https://github.com/arihanthsharma15/CliniqAI/tree/main/docs)) — Deterministic FSM voice assistant for patient booking, refill, and callback calls over Twilio. 3.9 average turns to resolution, 11.8% escalation rate, zero missed emergency escalations across simulated runs.

**[SOAPify](https://github.com/arihanthsharma15/soapify)** — Async RAG pipeline generating structured clinical SOAP notes in under 2 seconds, replacing a 12–16 minute manual process. Doctor-scoped vector retrieval prevents cross-user data leakage.
